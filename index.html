✦ V I E W P R O J E C T ✦ V I E W P R O J E C T
Sales Performance Analysis Using AdventureWorksDW2022
In this project, I explored the AdventureWorksDW2022 data warehouse to simulate a real-world business intelligence scenario for a global retail company. Using Microsoft SQL Server Management Studio and advanced Transact-SQL queries, I analyzed reseller sales data across various dimensions such as time, geography, product categories, and customer segments. Key insights included that the top 10 best-selling products generated 35% of total revenue, regional sales were led by the US and Europe, seasonal peaks consistently occurred in Q3, and shipping delays were linked to reduced customer satisfaction in certain areas.

Sentiment Analysis
Developed a deep learning-based sentiment analysis model using Keras to classify user-generated text as positive, negative, or neutral. I trained an LSTM-based neural network on a labeled dataset after preprocessing the text with tokenization, padding, and word embeddings. The final model was saved and deployed through a Python interface that allows real-time sentiment predictions with a simple user input via CLI or GUI.
